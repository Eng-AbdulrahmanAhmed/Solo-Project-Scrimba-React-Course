# Quizzical

A quiz app built with **React**, **TypeScript**, and **Vite**.

## Features

- Start screen with animated blobs
- 10 random trivia questions per game (from Open Trivia DB)
- Multiple choice answers shuffled randomly
- Answer highlighting (correct/wrong) after submission
- Score display with Play Again functionality

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
